# 高级布局选项 (Layout)

`fe-auto-factory` 支持内置常见业务场景中所需的页面骨架：

## `layout: admin`

中后台系统常用的**管理面板页面**，通过传入该布局，脚手架会直接构建带用侧边导航栏及顶部面包屑导航的 `Admin UI` 框架，而不再是空空白板。

## `layout: dashboard`

仪表盘布局框架。自动渲染带有 `aside` 及主展示面板区的二栏分离骨架。

## `layout: blank` (默认)

如果你仅需要一个单纯的全屏 H5 单页或是无基础侧边栏组件的展示，不传入或显式使用 `blank` 即可获得最干净透明的代码骨架层。
